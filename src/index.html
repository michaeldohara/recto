Make Recto your default reader?

Recto isn't currently the default opener for these file types. Setting it as default lets you double-click these files in File Explorer to open them in Recto.

    Windows only lets you set defaults from Settings. We'll open the Default Apps page and copy Recto to your clipboard — paste it in the search box at the top, click Recto, then set the file types you want.
